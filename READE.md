 # MINI CALCULATOR 

A simple modular CLI calculator buit to practice python functions, error handling, and file I/o. 

## Features 
-Addition, subtraction, multiplication, division, power, modulo
-Division/modulo by zero handled gracefully 
-Session history saved to  `history.txt`
-Option to view or clear previous history 

# How to run
1. Ensure you have python 3 installed.
2. Run the program:

```bash
python main.py
